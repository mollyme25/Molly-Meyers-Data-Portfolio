# Healthcare Anomaly Detection & Policy Audit
**Tools:** R, RMarkdown, Unsupervised Learning

### 🎯 The Objective
Developed an automated audit framework to identify rare, high-risk anomalies within clinical datasets. The primary focus was on "sensitivity" (catching every potential error) to mitigate ethical and operational risks in healthcare policy.

### 📈 Key Results
* **100% Sensitivity:** Engineered a model that identified 100% of high-risk outliers, ensuring no critical anomalies were missed during the audit process.
* **Risk Mitigation:** Prioritized ethical data handling by creating a framework that differentiates between standard statistical variance and actual clinical errors.
* **Technical Depth:** Utilized RMarkdown to create a fully reproducible report, documenting the logic from raw data ingestion to final risk assessment.
