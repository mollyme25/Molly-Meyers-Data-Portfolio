The Mission: As the Data-Driven Wine Connoisseur at Blythefield Country Club, I developed a chemical-based classification system to replace subjective tasting notes with objective data.

Key Achievements:

Clustering: Identified 4 distinct wine profiles (Dry Reds, Dry Whites, Sweet Whites, and Light Reds) using K-Means Clustering to automate member recommendations.

Accuracy: Achieved a 97.8% classification accuracy using a Distance-from-Center method, significantly outperforming traditional Decision Tree models (89%).

Business Impact: Enabled precision "look-alike" recommendations, allowing the club to suggest chemically similar wines to members with 97%+ confidence.
